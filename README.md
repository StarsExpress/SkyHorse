# 🏇天码行空 SkyHorse

## 仓库的前身
![repo_history.png](static/img/repo_history.png)

原本在2024年初 我开仓库只是想

储存所有写过的数据结构和演算法练习代码

当时作为数据结构和演算法的入门初学者

眼前还是比较疑惑的 不确定自己能否熬过来

## 仓库的路程
__两年一下就过去了__ 随著答对的LeetCode题数破480

AC的Hard题总数冲上200+大关

我对于数据结构和演算法 慢慢地有些不同的感觉

有不少时候 平时好动又善说话的我

竟能不知不觉间 脑中眼里只剩下LeetCode难题

再不然就是尝试能否AC得更快速 代码架构弄得更简洁

非常难解释清楚这究竟是什么状态

只能说我知道内化来出如此的面向

## [仓库的如今](https://starsexpress.github.io/SkyHorse/)
