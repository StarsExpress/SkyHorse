## 努力前行 远在天边 积少成多


## 仓库的前身
![repo_history.png](static/img/repo_history.png)

原本在2024年初 我开仓库只是想

储存所有写过的数据结构和演算法练习代码

当时作为数据结构和演算法的入门初学者

眼前还是比较疑惑的 不确定自己能否熬过来

尤其看Tim Roughgarden教授的网课满烧脑的🧠


## 仓库的路程
两年一下就过去了

随著答对的LeetCode题数破480

AC的Hard题总数冲上200+大关

我对于数据结构和演算法 慢慢地有些不同的感觉

曾经我觉得自己只爱玩 不喜欢动脑筋

不知不觉间 如今竟能有时候脑中眼里

专注于答对还没碰过的LeetCode难题

再不然就是尝试能否AC得更快速

代码架构弄得更简洁

非常难解释清楚这究竟是什么状态

可我清楚 如此面向 完全成为我的一部分了


## [仓库的如今](https://starsexpress.github.io/SkyHorse/)
有一天 我无意间在仓库查所有历史

来到200+个Commits 心想『干嘛停在这儿呢』

好多时候 这些思绪其实都是我后知后觉意识到的

虽然来得有点晚 但或许正是因此
